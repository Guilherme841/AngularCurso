DIRETIVAS ESTRUTURAIS

Diretivas Estruturais - Interagem com a view e modifica o DOM ou código HTML.

Componentes são diretivas com Template.

*ngFor - é uma diretiva(loop).

*ngIf - é uma diretiva de condição(if & else)

[hidden] - Também é uma opção, ele deixa o elemento no dom ocultado, mas ainda é visível no devTools, enquanto ngIf o remove. "hidden" mais apropriado em casos de dinâmica, porque age como um cache de elemento e mais propícios a alertas simples(árvores pequenas). ngIf se sai melhor com árvores grandes.
