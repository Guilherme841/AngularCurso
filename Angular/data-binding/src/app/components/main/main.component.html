INTERPOLAÇÂO

Interpolação são feitas abrindo duas vezes chaves, podem ser usadas para chamar funções,mostrar dado de uma variavel, executar algo e fazer alguma condição , por exemplo se variavel for true executar chamar alguma função.

Soma com interpolação: {{ 3.5 + 3.5 }}

Soma com função interpolada: {{ 1 + 1 + getValor() }}

Fazer pedido: {{ order && getOrder() }}

Property binding

Property binding é colocado uma propriedade do elemento dentro de colchetes , sendo assim posso adicionar um valor de uma variavel a propriedade dentro do binding sem interporlação e fazer a atualização dos dados exibidos, posso fazer condições. Exemplo: [src]'="imagemVariavel ? 'anotherImage' : 'anotherImage'".
