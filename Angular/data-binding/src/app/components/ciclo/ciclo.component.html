Documentação do Angular sobre lifeCycles: LifeCycles

LifeCycle é um estágio do componente desde sua criação até sua destruição, existem métodos que serão executados conforme o estágio que o componente se encontra, posso tipar esses métodos com "void" para não retornar valor.

  • ngOnInit() - é uma função que será executada sempre que o componente é inicializado/exibido no template, geralmente usado para fazer a chamada do servidor para obter dados e mostrar no template.
  • onChange() - é uma função que será executada sempre que algum dado de uma propriedade vinculado ao componente for alterada, geralmente usada com inputs porperties.
